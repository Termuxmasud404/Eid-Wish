📖 About Project

This is a beautiful animated Eid Mubarak wishing website created using HTML, CSS and JavaScript.
The website gives users a magical Islamic greeting experience with premium animations, glowing effects, floating Islamic symbols, changing Eid wishes, visitor interaction and modern UI design.

This project was specially designed to create a joyful and emotional Eid atmosphere for everyone.
